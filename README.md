# 혐오 탐지 프로젝트 (Hate text Detection)

## 📌 프로젝트 소개
본 프로젝트는 빅데이터를 활용하여 영상 및 텍스트 콘텐츠 내의 혐오 표현을 탐지하는 모델을 개발하는 프로젝트입니다. 

## 🚀 주요 기능
- **데이터 처리**: 텍스트 분석
- **모델 비교**: BERT와 RoBERTa를 활용한 혐오 표현 분류 성능 비교
